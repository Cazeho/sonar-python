Useless parentheses can sometimes be misleading and so should be removed.

The following code snippet illustrates this rule:

return 3             # Compliant
return (x)           # Non-Compliant
return (x + 1)       # Non-Compliant
x = (y / 2 + 1)      # Non-Compliant
y = (4+X) * y;       # Compliant
